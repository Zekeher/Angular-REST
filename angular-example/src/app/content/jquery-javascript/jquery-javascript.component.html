Usar Jquery & JavaScript en Angular

En este componente vas a aprender como usar Jquery y JavaScript en Angular

  • Necesitamos instalar las siguientes independecias:
    1. npm install --save jquery
    2. npm install --save @types/jquery
  • Luego declaramos la libreria en angular-cli.json en la parte "scripts".
