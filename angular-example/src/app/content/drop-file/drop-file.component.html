Drop & drag Files

En este componente vas a aprender como hacer un Drop & drag files

  • Necesitamos instalar las siguientes independecias:
    1. ng-drag-drop
    2. ng2-file-upload
  • Luego creamos una directiva para el drop & drag (en este caso lo llamamos file-drop.directive.ts).
    Creamos la directive con el siguiente comando "ng g d file-drop"
  • Luego creamos o utilizamos un componente que contenga dicha directive.
  • Opctional: Tambien podemos crear un models de File por si queremos que tenga una arquitectura especifica.

Creamos un Componente para llamar la directive del Drop & Drag File.
