Instrucciones para Modal de Angular

Cuando tengamos nuestra carpeta modal-angular en nuestro proyecto hay que hacer lo siguiente:

  1. Declarar el componente modal-angular en app.module.ts dentro de declarations
  2. Luego en el componente donde queremos mostrar el modal hacemos lo siguiente:
    • Primero creamos una variable por default OBLIGATORIA!!!
      No se tiene que modificar nada!!!
    • Se tiene que crear la siguiente funcion para que se cierre el modal OBLIGATORIO!!!
      No se tiene que modificar nada!!!
    • Despues creamos una funcion para setear los valores que queremos que muestre el modal, esta funcion va ser llamada en un button, link o lo que sea cuando hagamos click con el mouse ( esto es un ejemplo )
    • Luego en el Html del componente correspondiente agregamos la etiqueta correspondiente.
      {{exampleTag}}
